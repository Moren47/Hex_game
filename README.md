# Hex_game
Implementation of simple Hex Board game using:
 - tkinter  
 - hexagonal grid 
 - pathfinding algorithm  
 - socket communication  
 - selectors  
# Requirements
Python 3.8.10+  
# Launch the game in one window:
$ pipenv run python server.py
# Run server:
$ pipenv run python server.py
# Run 3 clients:
$ pipenv run python client.py 3
